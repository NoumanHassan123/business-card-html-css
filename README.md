# Business Card — HTML & CSS

A responsive digital business card built while learning full-stack development from the ground up. Displays a profile photo, name, title, and contact details in a clean card layout.

## Tech Stack
- HTML5
- CSS3 (Flexbox)

## What I practiced
- Structuring content with semantic HTML tags
- Centering and aligning elements using Flexbox
- Styling a card component (shadows, borders, spacing)
- Linking local images and stylesheets correctly
